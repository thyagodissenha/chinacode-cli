# Changelog

All notable changes to this project will be documented in this file.

This project follows semantic versioning where possible.

## [0.1.0] - 2026-05-25

### Added

- Initial npm package metadata for the `chinacode` CLI.
- Global CLI entrypoint through the `chinacode` binary.
- `npx chinacode` usage path after the package is published to npm.
- Initial documentation for installation, quick start, configuration, slash commands, tools, and npm publishing.

### Notes

- This release targets Node.js 20 or newer.
- npm publication must be performed manually by a maintainer after running the release checklist.
