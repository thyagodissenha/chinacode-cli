# ChinaCode CLI Public Launch

This folder contains the Epic 4 hard launch plan for ChinaCode CLI.

## Launch Goal

Earn 500+ GitHub stars in the first 7 days after the public hard launch while converting early attention into durable community channels, issue feedback, npm installs, and newsletter coverage.

## Required Placeholders

Do not publish with placeholders still present. Replace every value below during launch readiness review:

- `[GITHUB_REPO_URL]`
- `[NPM_PACKAGE_URL]`
- `[DOCS_URL]`
- `[DEMO_GIF_URL]`
- `[DEMO_VIDEO_URL]`
- `[BLOG_ANNOUNCEMENT_URL]`
- `[DISCORD_OR_TELEGRAM_URL]`
- `[BENCHMARKS_URL]`
- `[MAINTAINER_EMAIL]`
- `[PRESS_KIT_URL]`
- `[LAUNCH_DATE]`
- `[LAUNCH_OWNER]`
