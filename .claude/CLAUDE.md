# CLAUDE.md

## Comandos Técnicos
- Build: `npm run build`
- Type-check: `npm run typecheck`
- Testes gerais: `npm run test`
- Testes unitários (rápidos): `npm run test:quick`

## Governança e Regras do Projeto
> [!IMPORTANT]
> Você deve ler, respeitar e seguir estritamente as regras de governança multiagente, padrões de issues do Linear e fluxos de QA definidos em [AGENTS.md](./AGENTS.md) na raiz do repositório antes de realizar qualquer tarefa de planejamento, especificação, implementação ou testes.
